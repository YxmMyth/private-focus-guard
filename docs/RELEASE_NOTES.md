# FocusGuard v3.0 发布说明

## 🎉 版本亮点

- ✨ **智能语义分析**：基于腾讯混元大模型，理解用户行为上下文
- 🎚️ **优先级层次系统**：当下对齐 > 历史趋势，避免误判死循环
- 🔄 **Recovery 状态识别**：用户回归工作后立即停止报警
- 🎯 **精确标签页关闭**：只关闭分心标签页，保留有用内容
- 💰 **虚拟货币经济**：通过专注货币激励自律行为
- 📊 **信任分体系**：动态调整干预策略，越自律越自由
- 🔒 **隐私保护**：数据本地存储，1 小时后自动挥发

---

## 📦 包含内容

### Windows 可执行文件
- `FocusGuard.exe` - 单一可执行文件（约 70MB）
- `bundled_config.env` - 内置配置（包含 API 密钥）
- `README.txt` - 快速使用指南

### 系统要求
- **操作系统**：Windows 10/11 64位
- **内存**：至少 4GB RAM
- **网络**：需要连接腾讯云 API
- **权限**：无需管理员权限

---

## 🚀 快速开始

1. **下载**：从 [Releases](https://github.com/YxmMyth/private-focus-guard/releases) 下载最新版本
2. **解压**：解压到任意目录
3. **运行**：双击 `FocusGuard.exe`
4. **使用**：点击"开始监控"按钮

**首次使用**：
- 程序会自动创建配置目录：`C:\Users\<用户名>\.focusguard\`
- 内置混元 API 密钥，无需额外配置
- 可通过"设置"按钮调整监控参数

---

## 🔧 配置选项

可通过设置界面调整：
- **窗口监控间隔**：默认 3 秒
- **监控检查间隔**：默认 30 秒
- **数据库路径**：默认 `~/.focusguard/focusguard.db`
- **日志级别**：默认 INFO

---

## ⚠️ 注意事项

1. **首次运行可能被 Windows Defender 拦截**
   - 这是误报，可以点击"允许"或"添加到排除列表"
   - 原因：PyInstaller 打包的exe可能被误识别

2. **API 密钥内置**
   - 本版本内置开发者的腾讯云 API 密钥
   - 产生费用由开发者承担
   - 如需使用自己的 API 密钥，请从源码编译

3. **数据隐私**
   - 所有监控数据存储在本地
   - 1 小时后自动删除旧数据
   - 不上传任何用户数据到服务器

---

## 🐛 已知问题

1. **Chrome 历史读取可能失败**
   - 原因：Chrome 正在运行时数据库被锁定
   - 解决：已自动处理，复制到临时文件读取

2. **无法关闭某些窗口**
   - 原因：权限不足
   - 解决：以管理员身份运行应用
