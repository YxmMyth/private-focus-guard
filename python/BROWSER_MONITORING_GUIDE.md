# 🎉 浏览器监控功能已完成！

## ✅ 新功能

现在FocusGuard不仅可以监控应用程序，**还可以监控浏览器访问的网站！**

### 🌐 支持的浏览器
- ✅ Google Chrome
- ✅ Microsoft Edge

### 📊 监控信息
- 应用程序名称（如 Chrome.exe, msedge.exe）
- 窗口标题
- **访问的URL**（新功能！）
- **网页标题**（新功能！）
- 访问时间

---

## 🚀 如何使用

### 1. 启动GUI应用
```
双击运行: python\run_gui.bat
```

### 2. 点击"启动监控"

### 3. 打开浏览器访问网站
- 打开Chrome或Edge
- 访问不同的网站
- 切换标签页

### 4. 查看实时显示
界面会显示：
```
📍 当前活动
应用程序: msedge.exe
窗口标题: 搜索结果 - Google - Edge
检测时间: 17:45:30

🌐 访问网站:
https://www.google.com/search?q=python
网页标题: python - Google 搜索
```

### 5. 查看活动历史表格
表格新增了 **URL列**，显示：
- 时间
- 应用名称
- 窗口标题/网页标题
- **URL（新增！）**
- PID

## 💡 实际应用场景

### 1. 监控学习/工作时是否分心
- 如果你应该在学习，但打开了B站/抖音
- 系统会记录访问时间和URL

### 2. 统计网站使用情况
- 查看最常访问的网站
- 分析时间分配

### 3. 查看历史记录
- 所有访问的网站都保存在数据库
- 可以查看任何时间段的活动

---

## 🔍 数据库中的URL记录

所有访问的URL都保存在：
```
C:\Users\你的用户名\.focusguard\data\focusguard.db
```

使用 **DB Browser for SQLite** 打开查看：
- 表名：`activities`
- 字段：`data` 字段包含URL信息（JSON格式）

## 📝 与任务管理器的区别

| 功能 | Windows任务管理器 | FocusGuard |
|------|-----------------|------------|
| 显示应用程序 | ✅ | ✅ |
| 显示窗口标题 | ❌ | ✅ |
| **显示访问的URL** | ❌ | ✅ **(独有)** |
| **历史记录** | ❌ | ✅ **(独有)** |
| **统计信息** | ❌ | ✅ **(独有)** |
| **数据库存储** | ❌ | ✅ **(独有)** |

---

## ⚠️ 注意事项

1. **隐私保护**：所有数据存储在本地，不上传云端
2. **浏览器需打开**：浏览器正在使用时才能读取历史
3. **延迟时间**：可能有5-15秒的延迟（浏览器写入History需要时间）
4. **仅支持HTTP/HTTPS**：不记录chrome://等内部页面
