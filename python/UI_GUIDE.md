# 🎉 FocusGuard 图形界面已启动成功！

## ✅ GUI界面功能

你的电脑上现在应该有一个 **FocusGuard** 窗口正在运行，包含以下功能：

### 📍 当前活动显示
- 实时显示当前活动窗口的应用名称
- 显示窗口标题
- 显示检测时间和进程PID

### 🎮 控制按钮
- **▶️ 启动监控** - 开始实时监控窗口切换
- **⏹️ 停止监控** - 停止监控

### 📊 统计信息
- 显示总活动记录数
- 显示数据库大小

### 📜 活动历史表格
- 显示最近20条活动记录
- 包括：时间、应用名称、窗口标题、进程ID

---

## 🚀 如何使用

### 1. 启动监控
点击 **"▶️ 启动监控"** 按钮

### 2. 测试监控
- 打开浏览器访问网站
- 切换到VSCode、Word等应用
- 观察界面上实时更新的活动信息

### 3. 查看历史
下方表格会自动更新，显示所有活动记录

### 4. 停止监控
点击 **"⏹️ 停止监控"** 按钮

---
